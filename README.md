# BlogApp
Blog with node + react + typescript + mongodb

## Back app
```git clone https://github.com/victormasson/BlogApp.git --branch backend```

## Run app
* Start project **PROD**
  * ```npm start```
* Start project **DEV**
  * ```npm run dev```
* Start test
  * ```npx jest```
  * ```npm jest --watch```
 * Start local mongo server with a *blog* database
